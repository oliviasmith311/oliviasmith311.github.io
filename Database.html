Music Database

In this project, I worked on a team to create a full-stack SPA using a RESTful API and JavaScript that catalogued musical artists along with a list of their albums and the songs on those albums. Users have the ability to add new artists, albums and songs to the database.

Skills used: RESTful API, Spring, Java, JavaScript, TDD, Agile (Scrum), JSON, RESTful APIs
