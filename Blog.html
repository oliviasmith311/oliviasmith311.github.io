Full-Stack Blog

In this project, I worked independently to create a full-stack blog for my favorite podcast, My Brother, My Brother and Me. On the site, a user can submit blog posts, which include an author, a category, and various hashtags of the users choice. The user can then search for other posts by author, category or hashtag.

Skills used: Java, Spring, MVC, HTML, CSS, TDD, Agile (Scrum), Source Control / Git
